# 🧮 Calculator Pro (Multi-language)

Современный инженерный калькулятор, написанный на **Rust** с использованием графической библиотеки **egui**. Быстрый, легкий и работает сразу на Windows и Linux.

---

## ✨ Основные возможности
* **Инженерные вычисления:** Поддержка тригонометрии, логарифмов и сложных выражений.
* **Мультиязычность:** Полная локализация на **Русский**, **Французский** и **Китайский** языки.
* **История вычислений:** Сохранение последних операций для быстрого доступа.
* **Адаптивный интерфейс:** Корректное отображение шрифтов (включая иероглифы) на всех системах.

---

## 🚀 Как запустить (Скачать)
Вам **не нужно** устанавливать Rust или Cargo, чтобы пользоваться программой. Просто скачайте готовый файл для вашей ОС:

1. Перейдите в раздел [**Releases**](https://github.com/DarkLineDIXX/Calculator-Free-Math/releases).
2. Скачайте последнюю версию:
   * **Windows:** `calculator-windows.exe`
   * **Linux:** `calculator-linux`
3. **Запуск в Windows:** Если появится окно "SmartScreen", нажмите *Подробнее* -> *Выполнить в любом случае*.
4. **Запуск в Linux:** Не забудьте дать файлу права на запуск: `chmod +x calculator-linux`.

## Примечание

Для корректного отображения многоязычного текста приложение автоматически пытается загрузить системный шрифт на Windows и Ubuntu.

- На Windows ищутся `C:\Windows\Fonts\msyh.ttc` и `C:\Windows\Fonts\arialuni.ttf`.
- На Ubuntu пробуются шрифты `Noto Sans` и `DejaVu Sans` в стандартных каталогах `/usr/share/fonts/truetype/`.

Если ни один из шрифтов не найден, приложение всё равно запустится, но для некоторых символов может потребоваться наличие системных шрифтов.
